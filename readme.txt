== Description ==

Conversion Blocks provides Gutenberg blocks designed specifically for conversion. Unlike most blocks that bind you to specific themes or page builders that come with bloat, Conversion Blocks works with ALL WordPress themes. Built natively for Gutenberg, every block is crafted for conversion with performance in mind. Stop using generic blocks that don't drive results. Use blocks that are built to turn visitors into customers.

### Blocks Included

* **Pricing Table** - Beautiful pricing displays with multiple plans
* **Accordion / FAQ** - Organize content without overwhelming visitors
* **Feature Grid** - Showcase features in a beautiful responsive grid with icons
* **Button** - Call-to-action buttons with hover effects

### Why Conversion Blocks?

* **Works with ALL Themes** - Unlike most blocks that bind you to specific themes, Conversion Blocks works seamlessly with any WordPress theme
* **No Bloat** - Built lean and clean, without the bloat many page builders bring
* **Pure Gutenberg** - Written natively for Gutenberg, no page builder needed, no lock-in
* **Conversion-Focused** - Every block designed with performance in mind to drive action and turn visitors into customers
* **Fast & Lightweight** - Optimized code for maximum performance
* **Beautiful Defaults** - Works perfectly out of the box
* **Mobile Responsive** - Looks great on all devices

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/conversion-blocks`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Find Conversion Blocks in the Gutenberg block inserter
4. Start building beautiful, high-converting pages!

== Frequently Asked Questions ==

= Do I need a page builder like Elementor? =

Nope! Conversion Blocks works natively with Gutenberg, the default WordPress editor. No page builder needed.

= Will this work with my theme? =

Yes! Conversion Blocks works with ALL WordPress themes that support Gutenberg. Unlike most blocks that bind you to specific themes, Conversion Blocks is designed to work seamlessly with any theme you choose.

= Is the free version really free forever? =

Absolutely! The free version includes 3 core blocks with no time limits or hidden fees.
